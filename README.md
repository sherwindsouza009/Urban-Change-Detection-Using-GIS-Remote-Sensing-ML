# Landsat Change Detection (2014–2024)

## Overview

This project analyzes urban expansion in Southern Ontario (Toronto and surrounding regions) using Landsat 8 satellite imagery from 2014 and 2024.

## Methods Used

* NDVI Change Detection
* Image Regression & Residual Analysis
* Random Forest Classification

## Tools & Libraries

* Python (Whitebox Workflows)
* QGIS
* Landsat 8 Data

## Key Findings

* Significant urban expansion in Milton, Brampton, and Oakville
* Loss of vegetation linked to industrial and residential growth
* Classification results show land-cover transitions over time

## Files

* `UrbanChangeDetection.ipynb` → Main analysis notebook
* `UrbanChangeDetection_Report.pdf` → Full report

## Limitations

* NDVI sensitive to seasonal variation
* Classification errors due to similar spectral signatures
